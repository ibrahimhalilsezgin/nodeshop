## 🔧 # Kurulum ve Çalıştırma

> npm run dev
### frontend
.env dosyası doldurulduktan sonra:
> npm install <br>
> npm run build <br>
> npm run preview

### cdn
.env dosyası doldurulduktan sonra:
> npm install <br>
> npm run dev
